# TicTacToe-Game
A two-player console Tic-Tac-Toe game written in Python.
# Tic-Tac-Toe Game Instructions

Prerequisites

You need Python 3 installed on your system.

Execution Steps

Download the Code: Clone this repository or download the ZIP file.

```
git clone https://github.com/DigitalDistillation/TicTacToe-Game.git
cd TicTacToe-Game 
```
Start the Game: The game will begin automatically because of the entry point in the code.

Using the Terminal: Run the main file directly from your command line:

python tic_tac_toe.py


Using IDLE: Open the tic_tac_toe.py file in IDLE and press F5 (or go to Run > Run Module). The game will start automatically in the IDLE Shell.

Play:

The board is represented by numbers 1 through 9.

When prompted (Player X move:), enter the number (1-9) corresponding to the square where you want to place your 'X' or 'O'.

Enjoy the game!
